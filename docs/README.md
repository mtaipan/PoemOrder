## Карта проекта за 30 секунд

- **Что это:** сайт-визитка автора стихов на заказ — публичная витрина + форма заявки + админка.
- **Стек:** Java 21, Spring Boot 4 (MVC + Security + Data JPA), PostgreSQL 16, Flyway, Thymeleaf, Docker.
- **Объём:** ~40 Java-классов, 10 миграций, ~20 шаблонов. Один сервис, одна БД, один администратор (или несколько).
- **Запуск:** `docker compose up --build` при заполненном `.env` (обязателен `APP_ADMINS`).
